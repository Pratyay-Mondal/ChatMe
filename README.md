# ChatMe 
### A real-time chat application with multiple user groups

## Used Technologies:

### Front-end: 
HTML, CSS, JavaScript, Vanilla.js

### Back-end: 
Node.js, Express.js

### Client-Server Communication: 
Socket.io


```
npm run dev
```

```
http://localhost:3000/
```
